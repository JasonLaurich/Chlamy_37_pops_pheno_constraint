# Processed data

This folder contains datasets used in the analyses for Laurich et al. (2026), "Widespread Pareto fronts constrain the evolution
of ecological limits in phytoplankton".

The folder includes both raw datasets obtained from previously published studies and derived datasets generated during the analysis 
pipeline. Scripts in scripts-R/ read from and write to this directory.

For convenience, raw datasets obtained from external studies and processed datasets generated during this analysis are stored in the 
same folder. Future versions of the repository may separate these into dedicated raw and processed data directories.

## File naming convention

Processed data files are numbered according to the order in which they are generated or used in the analysis pipeline.

## File breakdown

### Raw data

Files containing raw growth data from Bernhardt et al., 2020:
  - 01_temp_rfus_time                    Files 00, 06, 10, 14, and 18 all contain the raw RFU data (column RFU), as well as
  - 06_light_rfus_time                     relevant data about the level of abiotic gradient (e.g. temperature),
  - 10_nitrogen_rfus_time                  time since the start of the experiment, and population data.    
  - 14_phosphorous_rfus_time
  - 18_salt_rfus_time 
  - 22_stoich_data                       Stoichiometric data
  - 23_id_mapping                        ID mapping file to convert between population labels used in 22 and the rest of the experiment
  - 24_pigment_data                      Pigment data
  - 25_cell_size                         Biovolume data
  - 26_labels_summary                    Full information on treatments for experimental evolution project

Files containing synthesis data for other phytoplankton species:
  - 28_Thomas_2012_raw_data              
  - 31_Bestion_2018_raw_data
  - 34_Lewington_Pearce_2019_raw_data
  - 38_Edwards_2016_raw_data
  - 41_Levasseur2025_l_rawdata
  - 42_Levasseur2025_n_rawdata
  - 43_Levasseur2025_p_rawdata
  - 66_Narwani_2015_summary 
  - 68_Edwards_2015_summary 
  
### Processed data
  - files 02, 07, 11, 15, 19             Estimates of intrinsic growth rate (µ) across abiotic gradients
  - files 04-05, 08-09, 
      12-13, 16-17, 20-21                Model estimates and fits for experimentally evolved C. reinhardtii populations
  - 27_summary_table                     Summary table compiling model estimates with stoichiometric, pigmentation, and biovolume data
  - files 29-30, 32-33, 35-37,
      39-40, 44-46                       Growth rates, model estimates and fits of thermal performance curves (TPCs) across phytoplankton
  - files 47-54                          Growth rates, model estimates and fits of Monod curves (light) across phytoplankton
  - files 55-60                          Growth rates, model estimates and fits of Monod curves (nitrogen) across phytoplankton
  - files 61-65                          Growth rates, model estimates and fits of Monod curves (phosphorous) across phytoplankton
  - files 69-72                          Summary data sets for comparative synthesis of phytoplankton niche-determining traits
  
All processed datasets in this folder can be regenerated by running the scripts in the scripts-R/ directory in numerical order.
